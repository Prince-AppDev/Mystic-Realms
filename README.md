> Eldoria from the forces of darkness!

---

## 🎮 About
Mystic Realms is a console RPG where you
play as a hero chosen to save the kingdom
of Eldoria. Battle monsters across 4
locations and complete the Village Elder's
quest!

---

## ✨ Features

| Feature | Description                 |
|---------|-----------------------------|
| 🗺️ Locations | 4 unique areas to explore   |
| 👹 Monsters | 4 different enemies         |
| ⚔️ Combat | Attack, Fireball, Lightning |
| 🧪 Healing | Potion system               |
| 📊 Stats | HP, XP, Attack              |
| 🎭 Story | Village Elder quest system  |

---

## 🚀 How to Run

### Option 1 - IntelliJ IDEA
1. Clone this repository
2. Open in IntelliJ IDEA
3. Run Main.kt ✅
### Option 2 - Kotlin Playground
1. Go to play.kotlinlang.org
2. Paste Main.kt code
3. Click Run ✅

## 🎯 Gameplay
Enter your hero name
Receive quest from Village Elder
Choose location to explore
Battle the monster!
→ Attack / Fireball / Lightning
→ Use Potion to heal
Win or lose — Eldoria's fate is yours!
